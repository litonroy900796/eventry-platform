# 🎫 Eventry Lite

## 🚀 Overview
Eventry Lite is a frontend-focused event discovery and ticket booking platform built with Next.js and MongoDB database integration.

The project supports **multi-language globalization (i18n)** to provide a better user experience for different users.

## ✨ Features
✅ Event listing and details page  
✅ User interest system  
✅ Ticket booking UI  
✅ MongoDB database integration  
✅ Multi-language support (Globalization)  
✅ Responsive modern design  
✅ Next.js optimized routing  

## 🛠️ Tech Stack
- Next.js  
- React  
- MongoDB  
- Tailwind CSS  
- i18n Globalization  

## 🌍 Multi-language Support
The project is designed with globalization support to handle multiple languages dynamically.
